Questions Fréquentes

Trouvez rapidement les réponses aux questions les plus courantes

Comment puis-je passer ma première commande ?

Après inscription, parcourez notre catalogue, sélectionnez vos produits et demandez un devis. Notre équipe vous accompagne dans tout le processus de commande.

Quelles sont les garanties qualité ?

Tous nos fournisseurs sont certifiés BPF/GMP. Chaque lot est accompagné de son certificat d'analyse (COA) et peut être testé par notre laboratoire partenaire.

Quels sont les délais de livraison ?

Les délais varient selon l'origine et le produit : 3-7 jours pour l'Europe, 10-21 jours pour l'Asie. Le suivi GPS est inclus pour toutes les expéditions.

Comment devenir fournisseur sur Chem Spot ?

Remplissez notre formulaire de candidature. Notre équipe d'audit évaluera vos certifications et installations avant validation. Le processus prend environ 2-4 semaines.

Y a-t-il des quantités minimales de commande ?

Les MOQ varient selon le fournisseur et le produit, généralement entre 1kg et 25kg. Ces informations sont indiquées sur chaque fiche produit.

Proposez-vous une API pour l'intégration ?

Oui, notre API REST complète permet l'intégration avec vos systèmes ERP/WMS. Documentation et support technique inclus pour les entreprises partenaires.
