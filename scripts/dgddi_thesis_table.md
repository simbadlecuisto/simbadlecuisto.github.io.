## Note méthodologique

Les données présentées dans ce tableau sont issues du portail statistique de la Direction Générale des Douanes et Droits Indirects (DGDDI), accessible via Lekiosque.finances.gouv.fr. Les statistiques douanières françaises sont établies selon la nomenclature combinée CN8 (NC8) de l'Union européenne, qui constitue la subdivision à 8 chiffres du Système Harmonisé. Les flux sont exprimés en valeur CAF (coût, assurance, fret) pour les importations.

**Codes valides avec données réelles (4/7 excipients)** : Lactose (17021100), Mannitol (29054300), Povidone/PVP (39053000), Croscarmellose Sodium (39123100), Cellulose Microcristalline/NOS (39129090), Stéarate de Magnésium (29159070).

**Code invalide avec données synthétiques (1/7 excipients)** : HPMC/Hypromellose (39123900 — HTTP 500 sur Lekiosque). Les données de substitution ont été estimées à partir des statistiques sectorielles des principaux acteurs mondiaux (Shin-Etsu Chemical, LOTTE Fine Chemical, Dow Chemical/Ashland).

**Remarque MCC** : Le code 39129090 (Cellulose et ses dérivés chimiques, n.d.a.) est un proxy plus précis que le code 17029090 (Autres sucres) précédemment utilisé. Il englobe les celluloses non spécifiées ailleurs, dont la MCC, mais exclut les acétates, nitrates et éthers de cellulose spécifiques. Le volume importé (23,1 M€, 2 883 tonnes) est notablement inférieur aux estimations antérieures, suggérant que la MCC pharmaceutique est largement produite localement ou importée sous d'autres codes.

**Remarque Stéarate de magnésium** : Le code 29159070 (acides monocarboxyliques saturés à l'exclusion des acides formique, acétique, propionique, butanoïque, pentanoïque, palmitique, stéarique, laurique...) constitue un proxy amélioré par rapport au code 29159090 initialement testé. La valeur élevée des importations (86,2 M€) reflète l'ensemble des acides gras saturés de cette sous-position, qui inclut les sels (dont le stéarate de magnésium) mais aussi d'autres dérivés industriels.

## Points clés de l'analyse

### Dépendance aux importations (7 excipients, 2023)
- **Valeur totale des importations : 251,0 M€** pour 75 928 tonnes
- **Stéarate de magnésium** : premier excipient importé en valeur (86,2 M€, 34,3% du total), données réelles code 29159070
- **Lactose** : deuxième excipient importé en valeur (62,1 M€, 24,7%), concentration européenne forte
- **Allemagne** : premier fournisseur global (lactose 53,7%, MCC 35,7%, PVP 42,6%, Croscarmellose 21,6%)
- **Dépendance asiatique limitée** : la Chine ne domine que pour le mannitol (34,7%) et l'HPMC (55,0%* synthétique)

### Concentration géographique et risque de rupture
- **Lactose** : concentration européenne forte (Allemagne 53,7%, Pays-Bas 19,3%) — **risque faible** grâce à la diversification intra-européenne
- **Povidone (PVP)** : dépendance à l'Allemagne (42,6%) et au Japon (17,4%) — **risque modéré** (alternative chinoise limitée à 9,0%)
- **Cellulose microcristalline** : dépendance Allemagne-Japon (67,4% cumulé) — **risque modéré-élevé** (deux sources majoritaires, approvisionnement très concentré)
- **Stéarate de magnésium** : dispersion européenne rassurante (Allemagne 22,6%, Pays-Bas 20,2%, Espagne 18,0%) — **risque modéré** mais code proxy imprécis
- **Croscarmellose Sodium** : forte dispersion européenne (11 fournisseurs, premier à 21,6%) — **risque faible**
- **HPMC / Hypromellose** : dépendance chinoise estimée à 55%* — **risque élevé** (données synthétiques, à confirmer)
- **Mannitol** : dépendance chinoise (34,7%) mais alternatives USA-Europe — **risque modéré**

### Particularité française : exportation de mannitol
La France est **exportateur net de mannitol** :
- **Importations 2023** : 7,8 M€ (1 538 tonnes)
- **Exportations 2023** : 106,9 M€ (11 545 tonnes)
- **Balance commerciale** : +99,1 M€ (excédent significatif)

Cette particularité s'explique par la présence de **Roquette Frères** (Lestrem, Hauts-de-France), leader mondial du mannitol pharmaceutique et alimentaire, qui exporte massivement vers les États-Unis (16,6%), l'Allemagne (11,7%), l'Espagne (10,0%) et l'Inde (7,5%). La France importe du mannitol chinois bas de gamme pour certaines applications, tout en exportant du mannitol haute pureté produit localement.

### Implications pour ChemistrySpot
- **Opportunité de sourcing alternatif** : la forte concentration géographique sur certains excipients (MCC : Allemagne/Japon 67,4%) crée un besoin de diversification
- **Différenciation qualité-prix** : l'écart de prix unitaire entre pays (ex. mannitol USA 12,3 k€/t vs Chine 3,1 k€/t ; MCC Japon 60,4 k€/t vs Chine 8,9 k€/t) justifie une plateforme offrant transparence et comparaison
- **Risque HPMC non quantifié** : l'absence de code NC8 valide pour l'HPMC est une lacune méthodologique significative — estimation 55% dépendance chinoise à surveiller
- **Stéarate de magnésium** : volume élevé (86,2 M€) mais code proxy (29159070) englobe d'autres acides gras — la part réelle du stéarate de magnésium pharmaceutique est probablement 30–40% du total
- **Complément données publiques** : les statistiques DGDDI fournissent une vision macroéconomique (flux pays), ChemistrySpot offre une vision microéconomique (fournisseurs individuels, certifications, délais)

## Limites méthodologiques

1. **Codes proxy NC8** : deux excipients (Povidone/39053000, Stéarate de magnésium/29159070) utilisent des codes élargis incluant potentiellement d'autres produits. L'HPMC (39123900) ne dispose d'aucune donnée douanière directe — estimation synthétique uniquement.

2. **HPMC non couvert** : le code 39123900 retourne HTTP 500 sur Lekiosque depuis au moins mars 2026. Cela peut indiquer une reclassification dans la nomenclature CN8 2025, une sous-position trop récente, ou une déclaration sous des codes génériques (39129090, 39139090). Des données alternatives (Eurostat Comext, UN Comtrade) pourraient pallier cette lacune.

3. **Seuil de confidentialité** : les flux bilatéraux inférieurs au seuil réglementaire ne sont pas publiés par la DGDDI. Cela peut expliquer l'absence de certains petits fournisseurs.

4. **Conversion monétaire** : les données DGDDI sont publiées en euros (CAF). Conversion en USD pour Supabase (taux moyen 2023 : 1,08 USD/EUR). Une variation de ±5% modifie légèrement les classements en valeur.

5. **Absence de distinction grade pharmaceutique / industriel** : les statistiques douanières ne distinguent pas les excipients de grade pharmaceutique (Ph. Eur., USP) des produits industriels. Notamment pour le stéarate de magnésium (utilisé industriellement) et la CMC/croscarmellose (applications alimentaires et industrielles).

6. **Réexportations** : certains flux comptabilisés en importation peuvent correspondre à des réexportations ultérieures (cas du mannitol notamment). Le volume de Pays-Bas pour le stéarate (5 416,7 t à 3,2 k€/t) suggère un trafic de négoce/distribution plutôt qu'une production locale.
